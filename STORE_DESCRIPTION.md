```
HoliduTracking is a powerful Chrome DevTools extension designed for developers and QA engineers working with Holidu's platform. It provides real-time insights into Holidu events, A/B tests, and tracking data directly within your browser's developer tools.

WHY INSTALL HOLIDUTRACKING?

- Real-time Event Monitoring: Instantly see all Holidu events (e.g., trackBatch API calls) as they occur, including generic events, custom events, and impression snapshots.

- A/B Test Visibility: Easily identify which A/B tests are active and their associated variants (BASE/NEW), helping you verify experiment setups.

- Comprehensive Tracking Data: Get a clear view of all tracking data being sent, ensuring data integrity and correct implementation.

- Network Request Insights: Monitor relevant network requests in real-time, providing context for your tracking data.

- Debugging & Validation: Quickly debug tracking issues, validate event payloads, and ensure your analytics are working as expected.

- User-Friendly Interface: A dedicated DevTools panel with intuitive search, filtering, and expandable views for easy navigation and analysis.

- Holidu-Themed Design: The extension features a clean, Holidu-branded interface for a consistent user experience.

KEY FEATURES:

- Event Tracking: Monitors trackBatch API calls and displays various event types (Generic, Custom, ImpressionSnapshot, A/B Test).

- Search & Filtering: Global search across all event fields with highlighting, and filtering by event type.

- Expandable Data: Truncates long values and allows expansion for detailed inspection.

- Clickable Offer Links: Directly navigate to offer pages from impression snapshot events.

- Error Detection: Highlights potential issues in event data (e.g., 404 errors).

PERFECT FOR:

- Frontend developers debugging tracking issues
- QA teams verifying event tracking
- Product managers analyzing A/B test results
- Anyone working with Holidu's tracking system

Install HoliduTracking to streamline your development and QA workflow, ensuring accurate and reliable tracking on the Holidu platform.
```
